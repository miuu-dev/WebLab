## 📖 Sobre este repositorio

Este repositorio reúne diferentes páginas y proyectos web que he ido desarrollando con el objetivo de **aprender, experimentar y mejorar mis habilidades en desarrollo Frontend**.

Cada proyecto representa una idea distinta: desde interfaces sencillas hasta diseños más completos, poniendo especial atención en el aspecto visual, la organización del código y las buenas prácticas.

Aunque la mayoría de los proyectos están construidos únicamente con **HTML** y **CSS**, algunos incluyen **JavaScript** para añadir interactividad cuando es necesario.

---

## 🎯 Objetivos

- Practicar desarrollo Frontend.
- Mejorar mis conocimientos de HTML y CSS.
- Experimentar con nuevos diseños y componentes.
- Crear interfaces modernas y responsivas.
- Tener un historial de evolución como desarrollador.

## 🚀 Filosofía del repositorio

No todos los proyectos tienen la intención de ser aplicaciones completas.

Muchos nacen simplemente como una forma de practicar una idea concreta, probar un diseño diferente o aprender una técnica nueva.

Este repositorio sirve como una especie de "laboratorio" donde puedo experimentar y seguir mejorando con cada proyecto.

---

## 📈 Evolución

A medida que vaya aprendiendo nuevas tecnologías y mejores prácticas, iré actualizando los proyectos existentes o añadiendo otros nuevos.

La idea es que este repositorio refleje mi crecimiento como desarrollador a lo largo del tiempo.
