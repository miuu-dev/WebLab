# 📊 Dashboard de Ventas

Este proyecto consiste en un **Dashboard de Ventas** desarrollado con **HTML, CSS y JavaScript**, cuyo objetivo es practicar la creación de interfaces modernas similares a las utilizadas en aplicaciones de administración reales.

He intentado recrear el diseño de un panel profesional, utilizando una estructura limpia, componentes reutilizables y gráficos interactivos para mostrar información de forma clara.

---

# ✨ Características

- 📌 Menú lateral de navegación (Sidebar).
- 📈 Gráfico de evolución de ventas (JavaScript).
- 🥧 Gráfico circular de ventas por categoría (JavaScript).
- 📊 Gráfico de barras por canal de venta (JavaScript).
- 📋 Tabla de productos más vendidos.
- 💳 Tarjetas con estadísticas principales.
- 🎨 Diseño moderno inspirado en dashboards profesionales.
- 📱 Estructura preparada para adaptarse fácilmente a responsive.

# 📄 Descripción de los archivos

## `dashboard.html`

Es la estructura principal de la página.

Contiene:

- Sidebar de navegación.
- Cabecera del dashboard.
- Tarjetas con estadísticas.
- Contenedores para los gráficos.
- Tabla de productos.

---

## `css/style.css`

Aquí se encuentra todo el diseño del proyecto.

Incluye:

- Variables CSS para los colores.
- Distribución mediante **CSS Grid**.
- Uso de **Flexbox** para alinear componentes.
- Diseño de las tarjetas.
- Estilos de la barra lateral.
- Estilos de los gráficos.
- Diseño de la tabla.
- Botones y efectos hover.

---

## `js/script.js`

Este archivo contiene toda la lógica relacionada con los gráficos.

Se utiliza la librería **Chart.js** para crear:

- Gráfico de líneas.
- Gráfico Doughnut.
- Gráfico de barras.

También se personalizan colores, escalas, animaciones y leyendas.

---

# 📁 Carpeta `images/icons`

Esta carpeta contiene todos los iconos utilizados en la interfaz.

Los iconos ayudan a identificar rápidamente cada apartado del menú lateral y algunos elementos de la página.

Incluye:
