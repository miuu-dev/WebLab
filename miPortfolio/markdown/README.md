# 🚀 Mi Portfolio

Este proyecto consiste en un **Portfolio Personal** desarrollado con **HTML, CSS y JavaScript**, cuyo objetivo es presentar de forma clara, moderna y profesional mi perfil, habilidades y proyectos como desarrolladora web.

He intentado recrear el diseño de una carta de presentación digital atractiva, utilizando una estructura limpia, componentes reutilizables y secciones interactivas para mostrar información de forma fluida.

---

# ✨ Características

- 📌 Secciones principales de presentación (Inicio, Sobre mí, Proyectos, Habilidades y Contacto).
- 🎨 Diseño moderno inspirado en interfaces profesionales y minimalistas.
- 📱 Estructura preparada para adaptarse perfectamente a dispositivos móviles y de escritorio (Responsive Design).
- 🖼️ Iconos personalizados y recursos gráficos organizados.
- 📄 Documentación integrada en formato Markdown.

# 📄 Descripción de los archivos

## `index.html`

Es la estructura principal de la página web.

Contiene:

- Cabecera y sección de bienvenida (Hero).
- Apartado de presentación personal y biografía.
- Sección de habilidades técnicas y herramientas.
- Galería o listado de proyectos destacados.
- Enlaces de contacto y redes sociales.

---

## `css/style.css`

Aquí se encuentra todo el diseño visual del portfolio.

Incluye:

- Variables CSS para la paleta de colores.
- Distribución mediante maquetación moderna.
- Uso de **Flexbox** y **CSS Grid** para la alineación de componentes.
- Estilos para tarjetas, botones y efectos de interacción (*hover*).
- Reglas multimedia (*media queries*) para garantizar la adaptabilidad responsiva.

---

# 📁 Carpeta `images/`

Esta carpeta contiene todos los recursos visuales e iconos utilizados en la interfaz del portfolio.

Incluye:

- **`icons/`**: Iconos de tecnologías (HTML, CSS, JS, Git), redes sociales (GitHub, LinkedIn, Instagram, Twitter) y elementos de navegación (usuario, carpeta, habilidades, logo).
- **`portada/`**: Recursos gráficos de previsualización y portadas de proyectos (como `portadaDashboard.png`).

---

# 🎯 Objetivos del proyecto

Con este proyecto he practicado:

- Organización y estructuración de carpetas en un proyecto web real.
- Uso de HTML semántico.
- Aplicación de CSS Grid y Flexbox.
- Uso de variables CSS para un mantenimiento limpio de los estilos.
- Diseño de interfaces limpias, accesibles y centradas en la experiencia de usuario.
- Buenas prácticas de control de versiones y despliegue continuo.

---

# 📚 Lo que he aprendido

Durante el desarrollo de este portfolio he aprendido a:

- Estructurar una carta de presentación digital profesional desde cero.
- Gestionar de forma eficiente recursos multimedia e iconos personalizados.
- Mejorar la responsividad y el diseño adaptativo en diferentes tamaños de pantalla.
- Mantener un código limpio, modular y fácil de escalar.
