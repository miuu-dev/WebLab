# 🚀 Mi Portfolio

> Carta de presentación digital y repositorio vivo de proyectos y aprendizajes en el desarrollo web.

---

## 🎯 Objetivo

**miPortfolio** es un proyecto personal diseñado para presentar de forma clara, moderna y profesional mi perfil, habilidades y proyectos como desarrolladora web. Su propósito principal es servir como escaparate para mostrar mi evolución técnica y profesional en el sector.

---

## 📂 ¿Qué incluye este repositorio?

* **Sitio web estático:** Páginas estructuradas en HTML5, estilos avanzados en CSS3 y comportamiento interactivo con JavaScript.
* **Documentación:** Archivos en Markdown para detallar la información y guías del proyecto.
* **Recursos multimedia:** Imágenes e iconos.

---

## 🛠️ Tecnologías y Herramientas

* **Lenguajes:** HTML5, CSS3, JavaScript (ES6+).
* **Control de versiones:** Git & GitHub.

---

## 📁 Estructura del Proyecto

```text
miPortfolio/
│
├── markdown/      # Documentación y archivos en Markdown
├── src/           # Código fuente del sitio (HTML, CSS, JS)
├── assets/        # Imágenes, logos y recursos multimedia
└── README.md      # Documentación principal del proyecto
```
